# SudokuSolver
Source files are in the 'src' folder:
### SudokuGui.java
 - Main class / GUI
 
### SudokuSolver.java
 - Logic class
 
### Running the program
 1. Compile both files in a same directory
 
 ```
 javac SudokuGui.java SudokuSolver.java
 ```
 
 2. Run SodukuGui
 
 ```
 java SudokuGui
 ```
